## Naming

Files were renamed with clear prefixes so duplicate names did not overwrite each other:

- `desktop_*` - desktop FastAPI web app files.
- `desktop_router_*` - desktop API route files.
- `backend_*` - full backend source files.
- `backend_router_*` - full backend API route files.
- `frontend_*` - Expo/React Native frontend files.
- `project_*` - project-level docs and deployment config.

The desktop app has been adjusted to run from this flat folder.
